Detailed Description

La clase Game_Over representa la ventana que se muestra cuando el juego termina.

Esta clase hereda de QGraphicsView y es responsable de mostrar la escena de Game Over. En esta ventana, el usuario puede elegir volver a la pantalla principal del juego.
