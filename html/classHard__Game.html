Detailed Description

Clase que define la ventana del juego en modo difícil.

Esta clase hereda de QGraphicsView y se encarga de crear la ventana del juego en modo difícil, donde el jugador tendrá menos balas y enemigos más resistentes y rápidos.

Member Function Documentation

◆ check_health

void Hard_Game::check_health ( )
slot

Método que revisa la salud del jugador para determinar si pierde o gana.

Este método es llamado por un QTimer cada 50 milisegundos para revisar si la salud del jugador llega a cero. Si la salud llega a cero, se muestra la ventana de Game Over y se cierra la ventana del juego. Si el jugador llega a la ronda 5 en la fase 2 con al menos una vida, se llama al método ganar() para mostrar la ventana de Victoria.

◆ decrease_bullets

void Hard_Game::decrease_bullets ( )
slot

Método que disminuye el número de balas del jugador.

Este método es llamado por un QTimer cada vez que el jugador dispara una bala. Si el número de balas llega a cero y el colector de balas está vacío, el QTimer se detiene.

◆ decrease_health

void Hard_Game::decrease_health ( )
slot

Método que disminuye la salud del jugador.

Este método es llamado por un QTimer cada vez que una nave enemiga colisiona con el jugador. Si la salud llega a cero, se llama al método check_health() para determinar si el jugador pierde o gana.
