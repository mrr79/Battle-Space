\doxysubsection{Detailed Description}
La clase \mbox{\hyperlink{classGame__Over}{Game\+\_\+\+Over}} representa la ventana que se muestra cuando el juego termina. 

Esta clase hereda de QGraphics\+View y es responsable de mostrar la escena de Game Over. En esta ventana, el usuario puede elegir volver a la pantalla principal del juego.
