\doxysubsection{Detailed Description}
Clase que define la ventana del juego en modo difícil. 

Esta clase hereda de QGraphics\+View y se encarga de crear la ventana del juego en modo difícil, donde el jugador tendrá menos balas y enemigos más resistentes y rápidos.

Método que revisa la salud del jugador para determinar si pierde o gana. 

Este método es llamado por un QTimer cada 50 milisegundos para revisar si la salud del jugador llega a cero. Si la salud llega a cero, se muestra la ventana de Game Over y se cierra la ventana del juego.
